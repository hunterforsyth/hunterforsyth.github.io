Galaxy Trader - Privacy Policy
Galaxy Trader uses Unity Analytics to collect some information about the game to help improve it, such as (but not limited to) when it is started for the first time, and if there are any crashes or errors. Absolutely no personally identifiable information is collected. You can opt-out at any time, or request a copy of this data, by opening the game and going to: Extras -> Privacy Policy.

For more information on Unity Analytics, please visit: https://unity3d.com/legal/privacy-policy.

If you have any questions about the privacy policy, please email: hunter@hforsyth.com.
