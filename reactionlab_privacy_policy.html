ReactionLab - Privacy Policy
ReactionLab Classic does not collect or transmit any data whatsoever.

ReactionLab 2 supports user generated content using the 'share' feature. At the user's request, a custom game scene can be uploaded. The scene also contains an upload date, scene name, and country tag. The scene can be removed from the server by reporting it (tap the flag icon), or by emailing hunter@hforsyth.com. Besides this, ReactionLab 2 does not store or transmit any data. At no point is any personally identifiable information captured, stored, or transmitted.

If you have any questions about the privacy policy, please email: hunter@hforsyth.com.
